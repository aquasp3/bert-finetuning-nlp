# NLP Token Classification with DistilBERT

## Project Overview
Fine-tuned a DistilBERT model for Part-of-Speech (POS) Tagging and Chunking using the CoNLL-2003 dataset as part of my Data Science Internship.

## Key Results
- **Training Loss:** 0.1137
- **Epochs:** 3
- **Training Time:** ~3 minutes (T4 GPU)
- **Metrics:** Evaluated using `seqeval` for sequence-accurate F1-scores.

## Technical Skills
- Subword Label Alignment
- Hugging Face Transformers & Trainer API
- Token Classification (Sequence Labeling)
- Python & PyTorch
